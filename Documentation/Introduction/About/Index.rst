================
What does it do?
================

This extension is a connector to the AdmiralCloud CDN interface.

This extension gives the user the ability to both upload files from the TYPO3 backend to AdmiralCloud as well
as select files previously uploaded to AdmiralCloud in TYPO3 content elements.
